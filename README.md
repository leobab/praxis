Este es el código de la plataforma web bolsapracticasutmach.com diseñada para la captacion de empresas y alumnos que deseen realizar practicas pre profesionales.

Empresas: Los encargados de ofertar puestos para practicantes.

Administrador: Es el que va a tener la función de validar todo
